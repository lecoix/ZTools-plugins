# plugin.json文件中配置路径说明
1. main、preload、logo的路径，均为dist目录相对于plugin.json文件的路径


# ZTools发布备注：
1. pull前开启Watt Toolkit，执行：git remote set-url origin https://github.com/honvin/ZTools-plugins.git


# utools发布步骤：
1. 先打包好项目，打包命令：npm run build
2. 将“utools打包文件”中的三个文件放到打包后的dist目录下
3. 删掉dist中用于调试的.js.map文件和.css.map文件，否则审核不通过


# 更新阿里巴巴图标库
1. 进入图标库项目管理：https://www.iconfont.cn/manage/index
2. 下载至本地，解压后复制css、ttf、woff、woff2 4个文件，进行替换
